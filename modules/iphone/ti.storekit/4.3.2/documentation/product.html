Ti.Storekit.Product

Description

A Ti.Storekit module object which represents a product from the in-app store.

Properties

Ti.Storekit.Product.titlestring

The title of the product.

Ti.Storekit.Product.descriptionstring

The description of the product.

Ti.Storekit.Product.pricedouble

The price of the product.

Ti.Storekit.Product.formattedPricestring

The price of the product, formatted for the store's locale.

Ti.Storekit.Product.introductoryPriceDiscount

The introductory price of the product, served as a Discount proxy. Includes the following properties: price (Number) priceLocale (String) subscriptionPeriod (Dictionary) { numberOfUnits (Number), unit (number) } numberOfPeriods (Number) * paymentMode (Number, DISCOUNT_PAYMENT_MODE_*)

Ti.Storekit.Product.subscriptionPeriodDictionary

The subscription period of the product, served as a dictionary with the following properties. numberOfUnits (Number) unit (Number, PERIOD_UNIT_*)

Ti.Storekit.Product.localestring

The locale of the product.

Ti.Storekit.Product.identifierstring

The product's identifier in the in-app store.

Ti.Storekit.Product.downloadableboolean

A boolean value that indicates whether the App Store has downloadable content for this product.

You can associate a set of data files with the iTunes Connect record you created for a product. The value of this property is true if at least one file has been associated with the product.

Ti.Storekit.Product.downloadContentLengthsarray<number>

The lengths of the downloadable files available for this product.

The array holds number objects, each of which holds value that is the size of one of the downloadable files (in bytes).

Ti.Storekit.Product.downloadContentVersionstring

A string that identifies which version of the content is available for download.

The version string is formatted as a series of integers separated by periods.
